Maybe API

Maybe is a data structure that models the presence or absence of a value. Maybe is analogous to the imperative style of null checks.

Full docs located here. 

Just :: a → Just a

Constructs a Maybe with a value.

import Maybe, { Just, Nothing } from "folktale/maybe"; Just(41); // Just(41) 
Nothing :: () → Nothing

Constructs a Maybe with no value.

import Maybe, { Just, Nothing } from "folktale/maybe"; Nothing(); // Nothing() 
.map :: (a → b) → Maybe b

Applies the supplied function to the value only if it's a Just. The function is ignored if applied to a Nothing.

import Maybe, { Just, Nothing } from "folktale/maybe"; Just(41).map(R.add(1)); // Just(42) Nothing().map(R.add(1)); // Nothing() 
.chain :: (a → Maybe b) → Maybe b

Applies the supplied function (that returns a Maybe) to the value and flattens the result only if it's a Just. The function is ignored if applied to a Nothing.

import Maybe, { Just, Nothing } from "folktale/maybe"; // nested Just when using .map Just(41).map(n => Just(n+1)); // Just(Just(42)) Just(41).chain(n => Just(n+1)); // Just(42) Nothing().chain(n => Just(n+1)); // Nothing() 
.getOrElse :: a → a

Returns the value from a Just, or a fallback .

import Maybe, { Just, Nothing } from "folktale/maybe"; Just(41).map(R.add(1)).getOrElse(0); // 42 Nothing().map(R.add(1)).getOrElse(0); // 0
