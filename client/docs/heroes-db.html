Heroes DB API

findAll :: () → Task [a]

Returns all heroes as a Task([Hero]).

import DB from "heroes-db"; const allHeroesTask = DB.findAll(); // Task([Hero]) allHeroesTask.run().listen({ onResolved: heroes => console.log(heroes), onRejected: error => console.log(error) }); 
findByName :: String → Task a

Takes a name and returns a Task(Hero). The task will be rejected if the hero cannot be found.

import DB from "heroes-db"; const superman = DB.findByName("Superman"); // Task.Resolved(Hero) const rejected = DB.findByName("Poopman"); // Task.Rejected 
query :: (a → bool) → Task [a]

Takes a predicate function and returns a Task([Hero]). All heroes that match the predicate will be returned If no heroes match, expect an empty list.

import DB from "heroes-db"; const superman = DB.query(R.propEq("name", "Superman")); // Task.Resolved([Hero]) 
heroes :: [Hero]

An array of Heroes.

import { heroes } from "heroes-db"; const superman = heroes[0];
