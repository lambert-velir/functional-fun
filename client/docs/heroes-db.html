    findByName :: String → Database → Task a

    Takes a "name" and the "database" and returns a Task(Hero). The task will be rejected if the hero cannot be found

    query :: (a → bool) → Database → Task [a]

    Takes a predicate function and the "database" and returns a Task([Hero]). All heroes that match the predicate will be returned If no heroes match, expect an empty list
