Box API

Box :: a → Box a

Constructs a box.

import Box from "box"; Box(41); // Box(41) 
.map :: (a → b) → Box b

Applies the supplied function to the value in the Box.

import Box from "box"; Box(41).map(R.add(1)); // Box(42) 
.unbox :: () → a

Returns the value from the Box.

import Box from "box"; Box(41).map(R.add(1)).unbox(); // 42
